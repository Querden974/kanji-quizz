# Kanji Quizz

Ce projet est une application React qui utilise Redux pour gérer les états et les actions.
Le but est de créer un jeu de quizz en japonais.

API utilisée : https://kanjiapi.dev/v1/

## Outils utilisés

- React
- Redux
- Vite
- Tailwind CSS
- Github
